\section{System Requirements}%
\label{sec:system_requirements}

Guten~Tag requires Glyphs version~3.0.3 or later with Python~3.8.0 or later.
Your Mac system version must be macOS~10.12 or later.

\paragraph{Update Glyphs}%
\label{par:update_glyphs}

Select \Path{Glyphs > Check for Updates…} from the menu bar.

\medbreak\noindent\Screenshot{Images/Glyphs > Check for Updates.png}

\paragraph{Update Python}%
\label{par:update_python}

Go to \Path{Glyphs > Preferences… > Addons} and check that your Python version is 3.8.0 or greater.

If no Python version is selected or it is less than 3.8.0, you can install Glyphs’ own Python:
Go to \Path{Window > Plugin Manager > Modules} and install Python.
Once Python has been installed, head back to the \Label{Addons} section in Glyphs’ preferences and choose the Python version that ends in \enquote{(Glyphs)}.
Relaunch Glyphs for the change to take effect.

\paragraph{macOS Version}%
\label{par:macos_version}

You can check the version of your macOS installation by selecting \Path{\AppleLogo > About This Mac}.
The version number is displayed underneath the macOS version name.
Guten~Tag works on macOS 10.12 \enquote{Sierra} or later.

\section{Install from the Plugin Manager}%
\label{sec:install_from_the_plugin_manager}

In Glyphs, open the Plugin Manager by selecting \Path{Window > Plugin Manager > Plugins}.
Search for \enquote{Guten Tag} and click \Label{Install} next to the plugin preview.
Relaunch Glyphs for the plugin to be loaded.
